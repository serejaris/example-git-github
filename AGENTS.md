# AGENTS.md · Colvir · занятие 7

Учебный репозиторий для практики Git и GitHub.

## Отчёт

- Презентация-отчёт живёт в одном файле: `index.html`
- Факты бери только из `brief.md`. Не выдумывай цифры и детали сверх брифа.

## Стили

- Подключай Tailwind только через CDN: `https://cdn.tailwindcss.com`
- Используй только utility-классы Tailwind
- Не добавляй `<style>`, отдельные `.css` файлы и inline `style="..."`

## Git

- Помогай с git-командами, если участник просит
- Сообщения коммитов — **на русском языке**
- Не клади в репозиторий секреты, токены и пароли
