# AGENTS.md · Colvir · занятие 7

## Не трогай `prompts/`

Папка `prompts/` — для ведущего и участников. **Не читай, не изменяй, не коммить** файлы из неё. Правила работы — в этом файле и `task.md`.

---

## Главное: выдумывай

Отчёт **полностью вымышленный**. Нет правильных цифр и фактов.

- Содержание слайда — по **промпту** своей задачи в `task.md` (раздел «Промпты»).
- **Придумывай** компанию, продукт, цифры, цитаты, конкурентов.
- Слайды **могут противоречить** друг другу — это часть игры.
- Будь креативным: юмор, драма, абсурд — ок, если вписывается в тип слайда.

---

## Критично: одна задача

1. Открой `task.md`. Найди свободную задачу и её **промпт** (таблица по номеру).
2. Возьми **только одну** задачу. Не бери вторую, пока первая не `готово`.
3. **Сразу отметь** в `task.md`:
   - статус → `в работе`
   - **Взял** → имя участника
   - **Агент** → своё имя
4. **Сразу скажи оператору вслух:**

   > «Беру задачу N — [название]. Роль: [роль]. Агент: [имя]. Оператор: [имя].»

5. Сделай слайд по промпту, статус `готово`, скажи:

   > «Задача N готова. Роль: [роль]. Агент: [имя]. Оператор: [имя].»

Без отметки в `task.md` и без сообщения оператору — работу не начинай.

---

## Подпись на слайде

Внизу слайда мелким серым текстом:

`Роль: [роль] · Оператор: [имя] · Агент: [имя]`

Классы Tailwind, например: `text-xs text-gray-400 mt-4`.

## Стили

- Tailwind только через CDN: `https://cdn.tailwindcss.com`
- Только utility-классы Tailwind
- Без `<style>`, без `.css`, без inline `style="..."`

## Порядок работы

1. Взять задачу и прочитать её промпт в `task.md`
2. Открыть `#slide-0N` в `index.html`
3. Скопировать вёрстку из шаблона (колонка «Шаблон»)
4. Заполнить слайд — **всё содержание придумать** по промпту
5. Подпись «Роль · Оператор · Агент» внизу
6. Статус `готово` в `task.md`
7. Коммит на русском, например: `Задача 7: главная цифра на слайде 7`
