# AGENTS.md · Colvir · занятие 7

## Критично: одна задача

1. Открой `task.md` и посмотри свободные задачи. Текст слайда — в колонке **Подсказка**, роль — в колонке **Роль**.
2. Возьми **только одну** задачу за раз. Не бери вторую, пока первая не `готово`.
3. **Сразу отметь** в `task.md`:
   - статус → `в работе`
   - **Взял** → имя участника (оператор)
   - **Агент** → своё имя (например `Cursor`, `Codex`, `Claude Code`, `ChatGPT`)
4. **Сразу скажи оператору вслух:**

   > «Беру задачу N — [название]. Роль: [роль из таблицы]. Агент: [твоё имя]. Оператор: [имя участника].»

5. Сделай слайд в `index.html`, поставь статус `готово` и снова скажи оператору:

   > «Задача N готова. Роль: [роль]. Агент: [имя]. Оператор: [имя].»

Без отметки в `task.md` и без сообщения оператору — работу не начинай.

## Подпись на слайде

Внизу слайда мелким серым текстом добавь строку атрибуции:

`Роль: [роль] · Оператор: [имя] · Агент: [имя]`

Используй Tailwind-классы, например: `text-xs text-gray-400 mt-4`.

## Отчёт

- Факты — `brief.md`
- Задачи — `task.md` (24 задачи, задача N → слайд N)

- Шаблоны вёрстки внизу страницы: `#templates` — **не перезаписывай**

## Стили

- Tailwind только через CDN: `https://cdn.tailwindcss.com`
- Только utility-классы Tailwind
- Без `<style>`, без `.css`, без inline `style="..."`

## Порядок работы

1. Взять задачу из `task.md` (см. блок «Критично» выше)
2. Открыть слайд из колонки «Слайд» в `task.md` (задача N = `#slide-0N`)
3. Скопировать вёрстку из подходящего шаблона (`#templates`)
4. Заполнить по колонке **Подсказка** в `task.md` (цифры сверить с `brief.md`)
5. Добавить подпись «Роль · Оператор · Агент» внизу слайда
6. Отметить задачу `готово` в `task.md`
7. Предложить коммит на русском, например: `Задача 7: продажи на слайде 7`
