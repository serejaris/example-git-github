# AGENTS.md · Colvir · занятие 7

## Критично: одна задача

1. Открой `task.md` и посмотри свободные задачи.
2. Возьми **только одну** задачу за раз. Не бери вторую, пока первая не `готово`.
3. **Сразу отметь** в `task.md`: статус `в работе`, в колонке «Взял» — имя участника.
4. **Сразу скажи оператору вслух:** «Беру задачу N — [название]».
5. После слайда в `index.html` поставь в `task.md` статус `готово` и снова скажи оператору.

Без отметки в `task.md` и без сообщения оператору — работу не начинай.

## Отчёт

- Презентация — `index.html`
- Факты — `brief.md`
- Задачи — `task.md` (24 задачи, задача N → слайд N)
- 24 пустых слайда: `#slide-01` … `#slide-24`
- Шаблоны вёрстки внизу страницы: `#templates` — **не перезаписывай**

## Стили

- Tailwind только через CDN: `https://cdn.tailwindcss.com`
- Только utility-классы Tailwind
- Без `<style>`, без `.css`, без inline `style="..."`

## Порядок работы

1. Взять задачу из `task.md` (см. блок «Критично» выше)
2. Открыть слайд из колонки «Слайд» в `task.md` (задача N = `#slide-0N`)
3. Скопировать вёрстку из подходящего шаблона (`#templates`)
4. Заполнить фактами из `brief.md`
5. Отметить задачу `готово` в `task.md`
6. Предложить коммит на русском, например: `Задача 2: главные цифры на слайде 5`

## Шаблоны слайдов

| ID | Название | Когда использовать |
|---|---|---|
| `template-cover` | Обложка | Титул, название, подзаголовок |
| `template-metrics` | Три цифры | Три KPI в ряд |
| `template-columns` | Две колонки | Плюсы и минусы |
| `template-cards` | Три карточки | Три темы одного уровня |
| `template-list` | Список | Перечень пунктов |
| `template-statement` | Один вывод | Вопрос или главная мысль |
