# Промпт для участников · занятие 7

Скопируйте текст ниже в чат с вашим AI-агентом (Cursor, Codex, Claude Code, ChatGPT — что используете).

---

```
Ты работаешь в учебном репозитории Colvir, занятие 7.

Прочитай AGENTS.md и task.md в корне проекта. Следуй им строго.

Твоя работа:
1. Найди одну свободную задачу в task.md (статус «свободна»).
2. Сразу отметь её: статус «в работе», в колонке «Взял» — моё имя (спроси, если не знаешь), в колонке «Агент» — своё имя.
3. Сразу скажи мне вслух: «Беру задачу N — [название]. Роль: [роль]. Агент: [твоё имя]. Оператор: [моё имя].»
4. Открой слайд из колонки «Слайд» в index.html.
5. Скопируй вёрстку из шаблона (колонка «Шаблон», блок #templates внизу страницы).
6. Заполни слайд по колонке «Подсказка» в task.md. Цифры сверь с brief.md — не меняй.
7. Внизу слайда добавь подпись: «Роль: … · Оператор: … · Агент: …» (мелкий серый текст).
8. Отметь задачу «готово» в task.md и скажи мне: «Задача N готова.»
9. Предложи коммит на русском и помоги с git push, если попрошу.

Правила:
- Только одна задача за раз. Чужие слайды не трогай.
- Стили только Tailwind CDN, без своих CSS.
- Блок #templates в index.html не перезаписывай.
- Роль из таблицы не меняй — она выдуманная, для игры.

Начинай: найди свободную задачу и возьми её.
```
