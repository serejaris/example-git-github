# Промпт для участников · занятие 7

Скопируйте текст ниже в чат с вашим AI-агентом.

---

```
Ты работаешь в учебном репозитории Colvir, занятие 7.

Прочитай AGENTS.md и task.md. Следуй им.

Это вымышленный квартальный отчёт. Правильных фактов нет — всё придумываешь ты.

Твоя работа:
1. Найди одну свободную задачу в task.md.
2. Прочитай промпт этой задачи (раздел «Промпты» в task.md).
3. Отметь: статус «в работе», «Взял» — моё имя (спроси), «Агент» — своё имя.
4. Скажи вслух: «Беру задачу N — [название]. Роль: [роль]. Агент: [имя]. Оператор: [имя].»
5. Открой слайд #slide-0N в index.html.
6. Скопируй шаблон из #templates (колонка «Шаблон»).
7. Заполни слайд по промпту — тексты и цифры выдумай сам, креативно.
8. Внизу слайда: «Роль: … · Оператор: … · Агент: …»
9. Статус «готово», скажи «Задача N готова», предложи коммит на русском.

Правила:
- Одна задача за раз. Чужие слайды не трогай.
- Tailwind CDN only. #templates не перезаписывай.
- Слайды могут не совпадать по фактам — это нормально.

Начинай: возьми свободную задачу.
```
