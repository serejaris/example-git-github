# brief.md · вымышленный квартальный отчёт

## Сценарий

Команда из **24 человек** собирает **квартальный отчёт о запуске продукта** — в одном `index.html`, 24 слайда.

Это **учебная игра**, не реальный бизнес. **Все факты выдуманные:** компания, продукт, цифры, конкуренты, отзывы, решения.

## Что важно

- У каждого участника — **своя задача** и **своя роль** (см. `task.md`).
- Агент читает **промпт задачи** и **придумывает содержание** слайда.
- Слайды могут **не сходиться** между собой — это нормально и смешно.
- Один файл, Tailwind CDN, шаблоны внизу `index.html`.

## Чего нет

- Нет правильных ответов.
- Нет общего брифа с цифрами, которые надо копировать.
- Нет проверки «соответствует ли соседнему слайду».
