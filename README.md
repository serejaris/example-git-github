# Учебный репозиторий · Colvir · занятие 7

Общий репозиторий для практики Git и GitHub на седьмом занятии.

**Отчёт на сайте:** https://serejaris.github.io/example-git-github/

После `git push` страница обновляется — команда видит общий результат.

## Что за файлы здесь лежат

Коротко, если вы не работали с git и репозиториями раньше.

| Файл | Простыми словами |
|---|---|
| `README.md` | Этот файл. Коротко объясняет, что это за репозиторий и что означают файлы в корне. |
| `AGENTS.md` | Правила для AI-агента: один `index.html`, стили только через Tailwind CDN. |
| `task.md` | 24 задачи — по одной на участника. У каждой своя **роль** (арт-директор, аналитик, бухгалтер…). Берёшь одну, отмечаешь себя и агента, говоришь оператору. |
| `prompt.md` | Готовый промпт для вставки в чат AI-агента — старт занятия. |
| `brief.md` | Бриф отчёта про запуск «Груша 19». Факты для слайдов. |
| `index.html` | Отчёт: 24 пустых слайда + шаблоны вёрстки внизу страницы. Публикуется на GitHub Pages. |
| `.gitignore` | Список файлов, которые git **не сохраняет** в общую историю. |
